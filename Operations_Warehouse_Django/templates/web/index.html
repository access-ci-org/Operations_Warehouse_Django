This RESTful API provides programmatic access to authoritative information about ACCESS infrastructure:

  • Resource descriptions
  • Resource integration roadmaps, badges, and related integration status.
  • Resource hosted software modules
  • Resource outages
  • Resource batch scheduler configuration
  • Resource queued and running jobs
  • Which projects have access to each ACCESS allocated resource

This API may be used by ACCESS projects, resource providers, or other ACCESS community developeds in higher level services and tools.

Browse API documentation using the above API Swagger and API Redoc links.

Information from this API is also viewable through WebApp and web pages listed in the:
   Operations Dashboard https://dashboard.operations.access-ci.org

Other special views

  • ACCESS Active Resources
  • ACCESS Fields of Science
  • Warehouse Processing Status
  • Warehouse Publishers
